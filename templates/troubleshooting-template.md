# 排障记录模板

## 基本信息

| 字段 | 内容 |
|------|------|
| **日期** | YYYY-MM-DD |
| **故障等级** | P0 / P1 / P2 / P3 |
| **影响范围** | 受影响的业务/系统 |
| **发现方式** | 监控告警 / 用户反馈 / 日常巡检 |
| **处理人** | |

## 故障现象

描述观察到的问题现象。

## 排查过程

### Step 1: 初步定位

- 检查的内容
- 发现的线索

### Step 2: 深入分析

- 进一步排查
- 关联信息

### Step 3: 根因确认

- 最终定位到的根本原因

## 根因

一句话概括根因。

## 解决方案

- 采取的措施
- 验证结果

## 后续改进

- 需要修复的系统性问题
- 新增监控/告警建议
- 流程改进建议

## 时间线

| 时间 | 事件 |
|------|------|
| HH:MM | 发现故障 |
| HH:MM | 开始排查 |
| HH:MM | 定位根因 |
| HH:MM | 实施修复 |
| HH:MM | 恢复正常 |
