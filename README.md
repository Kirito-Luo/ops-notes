# ops-notes 🐕

> 运维知识库 — 排障经验、技术笔记、工具备忘

## 目录结构

```
ops-notes/
├── troubleshooting/    # 排障经验
├── daily-notes/        # 日常运维笔记
├── tools-reference/    # 工具 & 命令备忘
├── templates/          # 记录模板
└── README.md
```

## 内容分类

### 🔧 troubleshooting — 排障经验

生产环境问题排查记录，包含现象、排查思路、根因、解决方案。

### 📝 daily-notes — 日常笔记

日常运维工作中积累的经验、踩坑记录、优化心得。

### ⚡ tools-reference — 工具备忘

常用命令速查、工具配置笔记、脚本片段。

### 📋 templates — 模板

排障记录模板、变更记录模板、巡检报告模板等标准化格式。

## 使用建议

- 排障时按 `templates/troubleshooting-template.md` 格式记录，方便复盘
- 每次变更前按 `templates/change-record-template.md` 留档
- 随手记的命令/技巧扔进 `tools-reference/`
